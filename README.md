# Menu app UI

⚠️ For cypress.io demo purpose only

Slides: https://cypress.slides.em-dev.org/

## How to start tests
```bash
# Install dependencies and generate env files
$ npm install
$ npm run gen:env

# Start api (both docker & docker-compose are required)
$ npm run api:start

# Start ui
$ npm run start:cy

# Open cypress
$ npm run cy:open

# OR run tests
$ npm run cy:run
```
